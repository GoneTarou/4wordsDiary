## 1. サービス概要

４文字日記は、日記をつけるのが面倒な人のためのアプリです。

日記をつけるのが面倒な人でも、ほんの少しの情報から過去の振り返りを行うことができることを目指します。

## 2. このアイデアはどこから生まれたか

### 2-1. きっかけとなった体験・感情

このアイデアの元になった
**自分自身の体験・感情・違和感**を教えてください。

RUNTEQ入学後、先輩方が作った日記アプリを使い始めました。しかし、いざ日記を書こうとすると、今日したことを思い出すのに時間がかかり億劫になっていきました。

### 10-3.キャッチアップ不足の懸念

キャッチアップ不足の段階では以下の懸念点があります。
1. **完成までのハードルが高い**
使用する言語やフレームワークが異なるため、キャッチアップにかかる時間は個人差があるものの、一定の時間を要することが予想されます。
2. **Issueの見積もりが難しい**
「どこまで理解していて、何が分からないのか」「調査にかかる時間」などが不明瞭な場合、開発期間内に完了することが難しくなってしまいます。
